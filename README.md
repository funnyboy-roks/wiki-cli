# Wiki CLI

Search Wikipedia and read summaries directly in your terminal!

![Screen capture animation](./wiki-cli.gif)

## Usage

```
# Install package globally
npm i -g @funnyboy_roks/wiki-cli

# Open the TUI
wiki

# View command usage
wiki -h

# Search using keywords
wiki [keywords]
```
